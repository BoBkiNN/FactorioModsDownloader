# FactorioModsDownloader

### Args:

\<unzip> \<customPath>

<> - not necessary

**unzip**

+unzip - unzip mods?  yes
-unzip - unzip mods?  no (default)

**customPath**

path to download mods

**Examples:**
unzip to D:/aboba :

java -jar filename.jar +unzip D:/aboba

only download to D:/aboba :

java -jar filename.jar -unzip D:/aboba

just unzip:

java -jar filename.jar +unzip

only download:

java -jar filename.jar
or dobleclick file
